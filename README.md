# PostList Dorm Room Checker

The purpose of PostList is to find all important information about a room and the associated resident easier. To maintain security and prohibit others from getting access to internal data the app reads a csv file with all necessary columns.
Necessary columns: ["Nachname", "Name", "Zimmer", "Department", "Role", "id"]

## Used Libraries
For reading the csv data I used papaparse.
